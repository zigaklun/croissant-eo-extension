Croissant EO Extension

Documentation Fields GitHub

Earth Observation
Extension

A specialized extension for the Croissant metadata format, designed for Earth Observation datasets with geospatial and temporal metadata.

Quick Start

{
  "@context": {
    "cr": "https://mlcommons.org/croissant/",
    "eo": "https://zigaklun.github.io/croissant-eo-extension/schema/v1.0.0/"
  },
  "@type": "sc:Dataset",
  "name": "my-eo-dataset",
  "eo:scope": "global",
  "eo:hasTask": ["classification", "segmentation"],
  "eo:spatialResolutionInMeters": 10.0,
  "eo:dataSource": ["Sentinel-2"]
}

Available Fields

eo:scope

Geographic coverage description

eo:hasTask

ML task description and type

eo:amountOfTrainingData

Number of training samples

eo:numberOfClasses

Number of classification classes

eo:band

Spectral band information with labels and wavelengths

eo:hasClass

Class definitions with labels and identifiers

eo:statisticsInfo

Dataset statistics and counts

eo:metricsInLIT

Performance metrics from literature

eo:labeling

Labeling methodology and annotator information

eo:spatialResolutionInMeters

Spatial resolution

eo:status

Dataset completion status
