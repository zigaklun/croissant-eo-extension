üöÄ Available Fields

  • eo:scope - Geographic coverage (global, regional, local)
  • eo:hasTask - Supported ML tasks
  • eo:spatialResolutionInMeters - Ground sampling distance
  • eo:dataSource - Source satellites/sensors
  • eo:amountOfTrainingData - Number of training samples
  • eo:numberOfClasses - Classification classes count
  • eo:status - Dataset status (complete, ongoing, deprecated)
  • eo:availability - Access level (public, restricted, private)
