Extending MLCommons Croissant for Earth Observation datasets

v1.0.0

üéØ Purpose

This extension adds Earth Observation-specific metadata fields to the MLCommons Croissant standard, enabling better description and discovery of satellite imagery datasets for machine learning.

üìã Quick Start

Add the EO namespace to your Croissant metadata:

                {
                "@context":
                {
                    "cr": "https://mlcommons.org/croissant/",
                    "eo": "https://zigaklun.github.io/croissant-eo-extension/schema/v1.0.0/"
                },
                "@type": "sc:Dataset",
                "name": "my-eo-dataset",
                "eo:scope":"global",
                "eo:hasTask": ["classification", "segmentation"],
                "eo:spatialResolutionInMeters": 10.0,
                "eo:dataSource":["Sentinel-2"]
                }

üöÄ Available Fields

  • eo:scope - Geographic coverage description
  • eo:has-task - ML task description and type
  • eo:amountOfTrainingData - Number of training samples
  • eo:numberOfClasses - Number of classification classes
  • eo:band - Spectral band information with labels and wavelengths
  • eo:has_class - Class definitions with labels and identifiers
  • eo:statisticsInfo - Dataset statistics and counts
  • eo:metricsInLIT - Performance metrics from literature
  • eo:labeling - Labeling methodology and annotator information
  • eo:spatialResolutionInMeters - Spatial resolution
  • eo:status - Dataset completion status
